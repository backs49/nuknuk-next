# 다음 기능 후보 검토 (2026-03-31)

## 현재 구현 완료 기능

- 상품 관리 (다중 이미지, 옵션, 상세 블록)
- 장바구니 + 다건 결제 (토스 페이먼츠)
- 주문 관리 (상태 변경, 메모, 결제 링크)
- 쿠폰/포인트 시스템 (자동 발급, 코드 쿠폰, 추천인)
- 혜택 미리보기 (상품 상세 페이지)
- 카카오 알림톡 (주문 완료 시)
- 관리자 대시보드 (매출 통계, 차트, 고객 관리)
- 상담 상품 (가격 숨김, 카카오 연결)

## 미구현 기능 후보

### 1. 리뷰 등록

| 항목 | 내용 |
|------|------|
| 고객 가치 | **높음** — 구매 결정에 가장 큰 영향. 실제 후기가 신뢰도 향상 |
| 관리자 가치 | **높음** — 사회적 증거(Social Proof), 재구매 유도, 상품 개선 피드백 |
| 구현 복잡도 | **중간** — 리뷰 CRUD, 이미지 첨부, 별점, 관리자 답글, 주문 연동 |
| 우선순위 | **1순위 추천** |

**구현 시 고려사항:**
- 주문 완료 고객만 리뷰 작성 가능 (주문번호 or 전화번호 인증)
- 별점 (1-5) + 텍스트 + 이미지 첨부
- 관리자 답글 기능
- 상품 상세 페이지 하단에 리뷰 섹션
- 포인트 적립 연동 (리뷰 작성 시 포인트 지급)

### 2. Q&A 기능

| 항목 | 내용 |
|------|------|
| 고객 가치 | **중간** — 카카오톡 채널로 대체 가능하지만, 다른 고객의 Q&A 열람 가치 있음 |
| 관리자 가치 | **낮음** — 카카오톡과 웹 Q&A 이중 관리 부담 |
| 구현 복잡도 | **중간** — 질문/답변 CRUD, 비밀글, 알림 |
| 우선순위 | **3순위** |

**구현 시 고려사항:**
- 카카오톡 채널과의 역할 분리 필요
- 공개/비밀 질문 옵션
- FAQ 섹션으로 대체하는 것이 더 효율적일 수 있음
- 소규모 운영 특성상 Q&A보다는 FAQ가 관리 부담 적음

### 3. 팝업 공지사항

| 항목 | 내용 |
|------|------|
| 고객 가치 | **낮음** — 대부분 즉시 닫기 클릭. UX 저해 요소 |
| 관리자 가치 | **중간** — 시즌 상품, 휴무일, 프로모션 안내 |
| 구현 복잡도 | **낮음** — 모달 + 관리자 설정 + 쿠키 기반 "오늘 하루 안보기" |
| 우선순위 | **2순위 (단, 형태 재검토 필요)** |

**트렌드 검토:**
- 전통적인 팝업 모달은 2020년대 중반 기준 **구식** 취급
- 대안: 상단 배너(띠 배너), 인라인 공지, 토스트 알림
- 쿠팡/네이버 등도 팝업 대신 상단 배너 + 인라인 프로모션 카드 사용
- **추천: 팝업 대신 "상단 공지 배너" 형태가 현재 트렌드에 부합**

### 4. 기타 잠재 후보

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 위시리스트/찜 | 관심 상품 저장 — 고객 로그인 필요 | 낮음 |
| 재고 관리 | 품절 표시, 수량 한정 — 예약제 특성상 우선순위 낮음 | 낮음 |
| 이메일 마케팅 | 주문 고객 대상 프로모션 — 카카오 알림톡으로 대체 가능 | 낮음 |
| 배송 추적 | 배송 상태 조회 — 소규모 운영에서는 과도 | 낮음 |

## 추천 구현 순서

1. **리뷰 시스템** — 고객 구매 전환율에 가장 직접적 영향
2. **상단 공지 배너** — 낮은 복잡도, 관리자 운영 편의 향상
3. **FAQ 섹션** — Q&A 대신 자주 묻는 질문 정리 (카카오 부담 감소)
